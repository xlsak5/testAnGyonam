webpack

webpack은 JavaScript 애플리케이션을 위한 정적 모듈 번들러 입니다.

웹팩(Webpack)은 모듈 번들러(Module Bundler)로서, 웹 애플리케이션을 구성하는 JavaScript, CSS, 이미지 등과 같은 정적인 에셋들을 모두 모듈로 간주하고 이를 관리해줍니다.

웹팩은 이러한 모듈들 간의 의존성을 분석하고 번들링하여 최적화된 정적 에셋을 생성합니다.

웹팩은 다양한 기능을 제공하여 개발자들이 모듈 기반의 웹 개발을 보다 쉽고 효율적으로 할 수 있도록 도와줍니다. 주요한 기능과 개념들을 살펴보겠습니다:

Webpack 설치


mkdir webpack
cd webpack

1.4 webpack.config.js 설정

npx webpack을 작성하면 파일이 번들링 됩니다.

해당 경로에 js파일이 생성되고. 코드가 1줄로 출력이 됩니다.


const path = require('path');

module.exports = {
    entry: {
        index: './src/index.js'
    },
    output: {
        path: path.resolve(__dirname, 'public'),
        // js/./src/index.js
        filename: "js/[name].js"
    }    
}

vite

Vite는 Vue.js 및 기타 프레임워크를 위한 웹 개발 도구로서, 개발 서버와 빌드 도구를 제공합니다. Vite는 개발 서버를 통해 빠른 환경에서 개발을 할 수 있도록 지원하며, 빌드 시에는 코드 번들링을 최소화하여 빌드 속도를 향상시킵니다.

vite 주소 : https://vitejs-kr.github.io/guide/


    mkdir 폴더명(원하는 경로에서 폴더 생성)
    cd 폴더명
    npm create vite@latest
    y
    원하는 코드 선택
    code . (비쥬얼 스튜디오로 실행 시키면 됨)
    package.json 의 script 수정을 안했으면 경로로 이동해서
    npm run build + start
