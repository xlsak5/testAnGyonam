01. 선언적 함수

선언적 함수는 매개변수, return값이 없는 함수 입니다.

{
    function func() {
        document.write("실행되었습니다.1");
    }
    func();    
}
결과 확인하기 실행되었습니다.1

선언적 함수

함수에 document.write()를 작성하고 함수를 호출합니다.

2. 익명 함수

이름이 없는 함수로, 함수를 정의하면서 함수 이름을 지정하지 않는 것을 말합니다. 일반적으로 함수를 정의할 때 함수 이름을 지정하여 함수를 호출하고 사용합니다. 하지만 익명함수는 이름 없이 함수를 정의할 수 있습니다.

{
    const func = function(){
        document.write("실행되었습니다.2");
    }
    func();
}
결과 확인하기 실행되었습니다.2

익명 함수

익명 함수는 변수에 데이터를 저장하고. 변수 이름으로 호출 합니다.

3. 매개변수 함수

입력값을 받아 출력값을 계산하는 함수(function)이지만, 추가적으로 매개변수(parameter)를 가지고 있어서 출력값을 계산하는데 영향을 줄 수 있는 함수를 말합니다. 

{
    function func(str){
        document.write(str);
    }
    func("실행되었습니다.3");
}
결과 확인하기 실행되었습니다.3

매개변수 함수

매개변수 함수는 함수에서 매개변수를 받고. 받은 매개변수를 활용합니다.

4. 리턴값(결과 / 끝) 함수

입력값을 받아 출력값을 계산하는 함수(function)이지만, 추가적으로 매개변수(parameter)를 가지고 있어서 출력값을 계산하는데 영향을 줄 수 있는 함수를 말합니다. 

{
    function func(){
        const str = "함수가 실행되었습니다.4";
        
        return str;
    }
    document.write(func());
}
결과 확인하기 함수가 실행되었습니다.4

리턴값 함수

함수 안에서 선언된 변수나 매개변수를 반환합니다.

05. 화살표 함수 : 선언적 함수

선언적 형태의 화살표 함수는 함수 이름을 지정하고 function 키워드를 사용하지 않습니다. 

{
    func = () => document.write("실행되었습니다.5"); 
    func();
}
결과 확인하기 함수가 실행되었습니다.5

화살표 함수 : 선언적 함수

함수에서 실행되었습니다.5를 출력하는 것입니다.

06. 화살표 함수 : 익명 함수

함수 이름을 지정하지 않습니다. 대신, 함수 표현식과 비슷하게 변수에 할당하여 사용합니다. 

{
    const func = () => {
        document.write("실행되었습니다.6"); 
    }
    func();
}
결과 확인하기 함수가 실행되었습니다.6

화살표 함수 : 익명 함수

상수에 함수의 값을 저장하고. 상수의 이름으로 함수를 호출합니다.

07. 화살표 함수 : 매개변수 함수

화살표 함수의 매개변수는 함수 선언에서 괄호 안에 선언됩니다. 

{
    func = (str) => {
        document.write(str); 
    }
    func("실행되었습니다.7");
}
결과 확인하기 함수가 실행되었습니다.7

화살표 함수 : 매개변수

함수 실행문에다가 데이터를 전달합니다. 그리고 함수는 데이터를 매개변수로 받아 실행합니다.

08. 화살표 함수 : 리턴값 함수

화살표 함수는 일반 함수와 마찬가지로 값을 반환할 수 있습니다. 

{
    func = () => {
        const str = "실행되었습니다.8";
        
        return str;
    }
    document.write(func());
}
결과 확인하기 실행되었습니다.8

화살표 함수 : 리턴값

함수 안에서 선언된 값이나 매개변수를 반환 합니다.

09. 화살표 함수 : 익명 함수 + 매개변수 + 리턴값

화살표 함수(Arrow Function)는 ES6(ECMAScript 2015)에서 추가된 함수 표현식입니다. 함수를 간단하게 정의할 수 있고, 코드를 더 간결하고 가독성 좋게 작성할 수 있습니다. 

{
    // 화살표 함수는 타입(let, const)을 적어 줘야합니다.
    const a = () => {
        document.write("실행1" + "
");
    }
    a();

    const b = () => {
        document.write("실행2" + "
");
    }
    b();

    const c = (str1, str2, str3) => {
        document.write(str1, str2, str3 + "
");
    }
    c("실행3", "실행4", "실행5");

    const d = (x, y, z) => {
        return x+y+z;
    }
    document.write(d(1,2,3)) + "
";
}
결과 확인하기 실행1
실행2
실행3, 실행4, 실행5
1, 2, 3

화살표 함수 : 익명 함수 + 매개변수 + 리턴값

선언 : 화살표 함수에 데이터를 저장하고 호출 합니다. 매개변수 : 화살표 함수에 값을 전달하고 그것들을 출려합니다. 리턴값 : 화살표 함수에서 값을 반환 합니다.

10. 화살표 함수 : 익명 함수 + 매개변수 + 리턴값 + 괄호 생략

{
    const func = str => {
        return str;
    }
    document.write(func("실행되었습니다."));

화살표 함수 : 익명 함수 + 매개변수 + 리턴값

일반 화살표 함수를 선언하고. 매개변수에 괄호를 없애고 선언하고 있습니다.

화살표 함수(Arrow Function)는 ES6(ECMAScript 2015)에서 추가된 함수 표현식입니다. 함수를 간단하게 정의할 수 있고, 코드를 더 간결하고 가독성 좋게 작성할 수 있습니다.

함수에 매개변수를 전달하고 반환값으로 출력합니다. (return값은 생략이 가능합니다.) (이렇게하면 this값으로 프로토타입을 생성할 수가 없습니다.)

13. 함수 유형 : 함수와 매개변수를 이용한 형태

함수(function)는 프로그래밍에서 작업을 수행하기 위한 코드의 블록입니다. 함수는 입력을 받아 처리하고, 결과를 반환합니다. 이 때 함수가 받는 입력을 매개변수(parameter)라고 합니다. 

{
    function func(num, str1, str2){
        document.write(num + "." + str1 + "가" + str2 + "되었습니다.
");
    }
    func("1", "함수", "실행");
    func("2", "자바스크립트", "실행");
    func("3", "리액트", "실행");
}
결과 확인하기 1.함수가 실행되었습니다.
2.자바스크립트가 실행되었습니다.
3.리액트가 실행되었습니다.

함수 유형 : 함수와 매개변수를 이용한 형태

함수 실행문에서 매개변수를 전달하고. 함수의 제어문에서는 매개변수를 받아서 출력을 해줍니다.

14. 함수 유형 : 함수와 변수를 이용한 형태

함수는 일련의 작업을 수행하는 코드 블록입니다. 함수는 입력을 받아들이고 출력을 반환할 수 있습니다. 함수는 중복 코드를 피하고 코드의 가독성과 유지 보수성을 향상시키는 데 도움이 됩니다. 함수는 프로그램의 기능을 모듈화하고 재사용 가능한 코드 조각으로 분할하는 데 사용됩니다. 

{
    function func(num, str1, str2){
        document.write(num + "." + str1 + "가" + str2 + "되었습니다.
");
    }
    const youNum1 = 1;
    const youNum2 = 2;
    const youNum3 = 3;
    const youStr1 = "함수";
    const youStr2 = "자바스크립트";
    const youStr3 = "리액트";
    const youCom1 = "실행";

    func(youNum1, youStr1, youCom1);
    func(youNum2, youStr2, youCom1);
    func(youNum3, youStr3, youCom1);
}
결과 확인하기 1.함수가 실행되었습니다.
2.자바스크립트가 실행되었습니다.
3.리액트가 실행되었습니다.

함수 유형 : 함수와 변수를 이용한 형태

변수에 데이터를 저장하고. 함수에 매개변수로 데이터를 전달합니다.

15. 함수 유형 : 함수와 배열을 이용한 형태

함수는 일련의 작업을 수행하는 코드 블록입니다. 함수는 입력을 받아들이고 출력을 반환할 수 있습니다. 함수는 중복 코드를 피하고 코드의 가독성과 유지 보수성을 향상시키는 데 도움이 됩니다. 함수는 프로그램의 기능을 모듈화하고 재사용 가능한 코드 조각으로 분할하는 데 사용됩니다. 

{
    function func(num, str1, str2){
        document.write(num + "." + str1 + "가" + str2 + "되었습니다.
");
    }
    const num = [1,2,3];
    const info = ["함수","자바스크립트","리액트","실행"];

    func(num[0], info[0], info[3]);
    func(num[1], info[1], info[3]);
    func(num[2], info[2], info[3]);
}
결과 확인하기 1.함수가 실행되었습니다.
2.자바스크립트가 실행되었습니다.
3.리액트가 실행되었습니다.

함수 유형 : 함수와 배열을 이용한 형태

배열에 데이터를 저장하고. 데이터의 인덱스 번호로 데이터를 호출하고 함수의 매개변수로 전달합니다.]

16. 함수 유형 : 함수와 배열, 객체를 이용한 형태

함수는 일련의 작업을 수행하는 코드 블록입니다. 함수는 입력을 받아들이고 출력을 반환할 수 있습니다. 함수는 중복 코드를 피하고 코드의 가독성과 유지 보수성을 향상시키는 데 도움이 됩니다. 함수는 프로그램의 기능을 모듈화하고 재사용 가능한 코드 조각으로 분할하는 데 사용됩니다. 

{
    function func(num, str1, str2){
        document.write(num + "." + str1 + "가" + str2 + "되었습니다.
");
    }
    const info = [
        {
            num1: 1,
            name: "함수",
            word: "실행"
        },
        {
            num2: 2,
            name: "자바스크립트",
            word: "실행"
        },
        {
            num3: 3,
            name: "리액트",
            word: "실행"
        }
    ];

    func(info[0].num1, info[0].name, info[0].word);
    func(info[1].num2, info[1].name, info[1].word);
    func(info[2].num3, info[2].name, info[2].word);
}
결과 확인하기 1.함수가 실행되었습니다.
2.자바스크립트가 실행되었습니다.
3.리액트가 실행되었습니다.

함수 유형 : 함수와 배열, 객체를 이용한 형태

26. 함수 종류 : 콜백 함수 : 함수를 두번 실행(다른 함수에 인수로 넘겨지는 함수)

콜백 함수(callback function)란, 다른 함수의 인자로 전달되어, 해당 함수 내부에서 호출되는 함수를 말합니다. 즉, 함수 내부에서 다른 함수를 호출할 때, 해당 함수를 매개변수로 전달하여 실행하는 것입니다. 

{
    function func(){
        document.write("2. 함수 실행");
    }
    function callback(str){
        document.write("1. 함수 실행");
        str();
    }
    callback(func);
}
