01. 퀴즈 이펙트 : 정답 확인하기 유형

정답 확인 버튼을 누르면 정답이 나오는 유형입니다.

  • 데이터 저장하기 : 변수
  • 요소 객체 : 메서드 : querySelector() : 문서에서 특정 CSS 선택자로 요소를 찾는 DOM 메서드입니다.
  • 요소 객체 : 메서드 : innerText : DOM 요소의 텍스트 내용을 가져오거나 설정하는 데 사용되는 속성입니다.
  • 요소 객체 : 메서드 : classList는 요소의 클래스를 추가하거나 제거하여 스타일을 설정하는데 사용됩니다.
  • 이벤트 객체 : 메서드 : addEventListener : JavaScript에서 이벤트를 처리하는 함수입니다.
  • element.style.속성 = "속성값" : 자바스크립트에서 CSS스타일을 표현하는 방법입니다.
  • addEventListener : JavaScript에서 이벤트를 처리하는 함수입니다.
